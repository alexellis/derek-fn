## derek-fn

### Derek badge

Example:

Add the following to your README:

```
[![Derek](https://alexellis.o6s.io/badge?repo=derek&owner=alexellis)](https://github.com/alexellis/derek/)
```

[![Derek](https://alexellis.o6s.io/badge?repo=derek&owner=alexellis)](https://github.com/alexellis/derek/)

Redirects are supported, where your .DEREK.yml points to another repository.
